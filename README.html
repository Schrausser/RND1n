Demo movie

Random Number Screen RND1n: Screensaver for Android platforms, which generates random number sequences (c.f. Schrausser, 1996, p. 1-8 ff., 2022b, p. 4 ff., res.) and displays them in a random order on the screen (s. Fig. 1 to Fig. 3). Both, the system's own and the SIGMA quasi-random function n are used, where

ni=a⋅rni−1ς−int(a⋅rni−1ς),

with restvalue rx, a=103, ς=3445. See sigma.inc, Schrausser (2022a, p. 17).
